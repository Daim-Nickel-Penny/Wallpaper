Wallpaper Demo

A jQuery plugin for smooth-scaling image and video backgrounds. Learn More.

Image

Create a new Wallpapered element by passing the source image file in the options object:

$("#element").wallpaper({
	source: "path/to/image.jpg"
});
<div id="element"></div>
Demo

Video

Create a video Wallpaper by simply defining proper source types:

$("#element").wallpaper({
	source: {
		mp4:  "path/to/video.mp4",
		ogg:  "path/to/video.ogv",
		webm: "path/to/video.webm"
	}
});
Demo

Play on Hover

Videos will autoplay and loop by default, but Wallpapers can also play on hover:

$("#element").wallpaper({
	hoverPlay: true,
	source: {
		mp4:  "path/to/video.mp4",
		ogg:  "path/to/video.ogv",
		webm: "path/to/video.webm"
	}
});
